What is Component Facotry in Angular? What is Practical Use of it? Please Provide Examples

Base class for a factory that can create a component dynamically. Instantiate a factory for a given type of component with resolveComponentFactory(). Use the resulting ComponentFactory.create() method to create a component of that type.

The ComponentFactoryResolver service exposes one primary method, resolveComponentFactory.

The resolveComponentFactory() method takes a component and returns a ComponentFactory.
