Child to Parent communication

When we want to pass the data from the child component to the parent component, we use input binding with @Output decorations. This approach is ideal when you want to share data changes that occur on things like button clicks, form entires, and other user events. In the parent, we create a function to receive the message and set it equal to the message variable.
