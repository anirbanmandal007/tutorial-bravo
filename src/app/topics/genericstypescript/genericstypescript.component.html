In TypeScript, Generics are basically a kind of tool that enables you to create reusable code components that work with a number of types instead of a single type.

The generic type parameter is specified in angular brackets after the name of the class. A generic class can have generic fields (member variables) or methods. In the above example, we created a generic class named KeyValuePair with a type variable in the angle brackets <T, U>
