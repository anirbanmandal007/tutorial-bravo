What is FromEvent in Angular? What is Practical Use of it? Please Provide Examples

"FromEvent" creates an observable sequence by adding an event listener to the matching DOMElement, jQuery element, Zepto Element, Angular element, Ember.js element or EventEmitter.

Signature:

fromEvent(target: EventTargetLike, eventName: string, selector: function): Observable

Arguments:
  1. target (Any): The DOMElement, NodeList, jQuery element, Zepto Element, Angular element, Ember.js element or EventEmitter to attach a listener. For Backbone.Marionette this would be the application or an EventAggregator object.
  2. eventName (String): The event name to attach the observable sequence.
  3. [selector] (Function): A selector which takes the arguments from the event emitter so that you can return a single object.
Example:
            
              import {{'{'}} fromEvent {{'}'}} from 'rxjs';
              const clicks = fromEvent(document, 'click');
              clicks.subscribe(x => console.log(x));

The above code binds an event listener to the document. When user clicks on the document, the "clicks" gets subscribed and pront the actual Event.

Returns:

An observable sequence of events from the specified element and the specified event.
