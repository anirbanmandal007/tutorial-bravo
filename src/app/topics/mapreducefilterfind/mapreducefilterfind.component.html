What is map/reduce/filter/find in angular? What is Practical Use of it? Please Provide Examples

  • map():The Map object holds key-value pairs and remembers the original insertion order of the keys. Any value (both objects and primitive values) may be used as either a key or a value.
  • reduce():reduce() method applies a function simultaneously against two values of the array (from left-to-right) as to reduce it to a single value.
  • filter():filter() method creates a new array with all elements that pass the test implemented by the provided function.
  • find():The find() method returns the value of the first element in the provided array that satisfies the provided testing function.
